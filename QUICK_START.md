# Quick Start Guide

## Getting Started with AI Disease Prediction System

### For New Users

#### 1. Create an Account (Optional)
Creating an account allows you to save your prediction history and access it from any device.

**Steps:**
1. Click **"Sign In"** button in the top right corner
2. Switch to the **"Sign Up"** tab
3. Enter your email address
4. Create a password (minimum 6 characters)
5. Confirm your password
6. Click **"Create Account"**

**Note:** The first person to register will automatically become an administrator!

#### 2. Use Without an Account
You can use the system without creating an account:
- Click **"Continue as Guest"** on the login page
- Or simply start using the home page
- Your history will be saved in your browser (localStorage)

### Using the System

#### Analyze Your Symptoms

1. **Go to Home Page**
   - You'll see a list of symptoms organized by category

2. **Select Your Symptoms**
   - Click on symptom badges to select them
   - Selected symptoms appear at the top
   - Use the search box to find specific symptoms quickly

3. **Get Prediction**
   - Click **"Analyze Symptoms"** button
   - View your results instantly

4. **Review Results**
   - See the predicted disease with confidence score
   - Read the disease description
   - Follow the precautions and recommendations
   - Check suggested medical tests
   - View alternative possible conditions

#### View Your History

**For Logged-In Users:**
1. Click your profile icon in the header
2. Select **"My Dashboard"**
3. View your statistics and complete history
4. Delete individual predictions if needed

**For Guest Users:**
1. Click **"History"** in the navigation
2. View your browser-stored history
3. Clear history if needed

### For Administrators

#### Access Admin Panel
1. Log in with your admin account
2. Click your profile icon
3. Select **"Admin Panel"**

#### Manage Users
- View all registered users

### Frequently Asked Questions

**Q: Do I need to create an account?**
A: No, you can use the system as a guest. However, creating an account allows you to save your history and access it from any device.

**Q: Is my data secure?**
A: Yes! For logged-in users, data is stored securely in Supabase with Row Level Security. Guest users' data stays in their browser.

**Q: How accurate are the predictions?**
A: The system uses advanced algorithms, but accuracy depends on the symptoms you select. Always verify with a healthcare professional.

**Q: Can I delete my history?**
A: Yes! Logged-in users can delete individual predictions from their dashboard. Guest users can clear their browser history.

**Q: What if I forgot my password?**
A: Password reset functionality will be added in a future update. For now, create a new account if needed.

**Q: How do I become an admin?**
A: The first registered user automatically becomes an admin. Existing admins can promote other users to admin role.
